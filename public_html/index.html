Название

Системное администрирование серверов на Linux, FreeBSD, WEB серверов на Nginx. Разрабатываю web приложения на Python/Django, Bootstrap3-4, VueJS2. Разрабатываю и сопровождаю базы данных MariaDB(MySQL) и PostgreSQL. Сопровождаю АТС на базе Asterisk. Разворачивал для себя сервер GIT.
